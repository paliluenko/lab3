# Портфоліо — Павло Аліліуєнко

Персональний портфоліо-сайт, створений у рамках **Лабораторної роботи №1** з дисципліни «Веб-технології» (викладач Годунов А.В., ХАІ).

## Опис

Багатосторінковий HTML5-сайт, що представляє мене як майбутнього фронтенд-розробника.
Реалізований виключно на HTML без CSS та JavaScript.

## Структура сайту

| Файл | Сторінка | Опис |
|------|----------|------|
| `index.html` | Головна | Вітальна сторінка з загальною інформацією |
| `about.html` | Про мене | Освіта, навички, таблиця технологій |
| `contacts.html` | Контакти | Контактна інформація та форма зворотного зв'язку |

## Технології

- HTML5 (семантична розмітка)
- Git / GitHub (контроль версій)

## Запуск

Відкрийте файл `index.html` у будь-якому браузері.
